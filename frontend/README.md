# KOSMOS Frontend

This is the Next.js 14 frontend for the KOSMOS Digital Agentic System.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Running

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open [http://localhost:3000](http://localhost:3000)

## API Integration

The frontend is configured to proxy API requests to `http://localhost:8000` (the Python API).
Ensure the API is running before using the chat.
