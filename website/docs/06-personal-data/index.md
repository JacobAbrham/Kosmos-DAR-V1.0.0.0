## Core Principles

### Data Sovereignty
**You own your data.** Full export capability is always available. No vendor lock-in.

### Local-First Processing
Sensitive data is processed on-device when possible. Cloud sync is opt-in.

### Privacy Zones

| Zone | Access | Example |
|------|--------|---------|
| **Public** | Shareable | Blog posts |
| **Professional** | Work colleagues | Project docs |
| **Personal** | User only | Private notes |
| **Sensitive** | Encrypted unlock | Medical records |
